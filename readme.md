# XDarts

XDart is a python based game. It is form of throwing game in which the computer decides the random location on your dart board. It is represented by "X" symbol on your dart board that is a 5x5 matrix. At the end of the game the user can know their score.

  - Userfrindly
  - Simple
  - Fun



### Tech

XDarts requires only python installation.
